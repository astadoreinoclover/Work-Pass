# Work-Pass

## Rodar o Front-end

Para rodar o projeto front-end, siga os passos abaixo:

```bash
# Navegue até o diretório do front-end
cd Front

# Entre no diretório do app
cd app

# Instale as dependências
npm i

# Inicie o servidor web
npm run web

```bash
